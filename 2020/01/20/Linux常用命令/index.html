Linux常用命令

发表于 2020-01-20 更新于 2020-04-29 分类于 Linux 阅读次数： Valine：

常用命令

Linux命令基本格式

命令 [选项] [参数]

注意

个别命令使用不遵循此格式
当有多个选项时，可以写在一起
简化选项与完整选项
-a 等于 --all

例如
ls [选项] [文件或目录]

-a：显示所有文件，包括隐藏文件
-l：显示详细信息
-d：查看目录属性
-h：人性化显示文件大小
-i：显示inode

文件权限

例如
-rw-r–r–

- 文件类型（- 文件  d 目录  l 软链接文件）
rw-    u所有者
r--    g所属组
r--    o其他人
r 读  w 写  x 执行

文件处理命令

目录处理命令

建立目录
mkdir -p [目录名] # make directory

-p  递归创建

切换所在目录
cd [目录] # change directory
简化操作

cd  ～    # 进入当前用户的家目录
cd    # 同上
cd  -    # 进入上次目录
cd  ..    # 进入上一级目录
cd  .    # 进入当前目录

查询所在目录位置
pwd # print working directory
删除空目录
rmdir [目录名] # remove empty directory
删除文件或目录
rm -rf [文件或目录] # remove

-r：删除目录
-f：不提示确认

复制命令
cp [选项] [源文件或目录] [目标目录] # copy

-r：复制目录
-p：连带文件属性复制
-d：若源文件是链接文件，则复制链接属性
-a：相当于 -pdr

剪切或重命名命令
mv [源文件或目录] [目标目录] # move
常用目录的作用

/  根目录
/bin  命令保存目录（普通用户就可以读取的命令）
/boot  启动目录，启动相关文件
/dev  设备文件保存目录
/etc  配置文件保存目录
/home  普通用户的家目录
/lib  系统库保存目录
/mnt  系统挂载目录
/media  挂载目录
/root  超级用户的家目录
/tmp  临时目录
/sbin  命令保存目录（超级用户）
/proc  直接写入内存的
/sys
/usr  系统软件资源目录
/usr/bin/  系统命令（普通用户）
/usr/sbin/  系统命令（超级用户）
/var  系统相关文档内容
proc和sys目录不能直接操作，这两个目录保存的是内存的挂载点
可以在家目录root或home，以及tmp目录下随便放内容
链接命令

ln -s [原文件] [目标文件] # link 生成链接文件

-s：创建软链接

硬链接特征

  1. 拥有相同的i节点和存储block块，可以看作是同一个文件
  2. 可通过i节点识别
  3. 不能跨分区
  4. 不能针对目录使用

软链接特征

  1. 软链接拥有自己的i节点和block块，但是数据块中只保存原文件的文件名和i节点号，并没有实际的文件数据
  2. 软链接文件权限都为rwxrwxrwx lrwxrwxrwx l表示软链接文件
  3. 删除原文件，软链接不能使用

文件搜索命令

locate命令格式

locate 文件名 # 在后台数据库中按文件名搜索，搜索速度更快
locate命令所搜索的后台数据库: /var/lib/mlicate
updatedb # 更新数据库
搜索遵守/etc/updatedb.conf配置文件

  • 是否开启搜索限制
  • 搜索时，不搜索的文件系统
  • 搜索时，不搜索的文件类型
  • 搜索时，不搜索的路径
搜索命令的命令whereis、which

whereis 命令名 # 搜索命令可执行文件所在路径及帮助文档所在位置

-b：只查找可执行文件
-m：只查找帮助文件

which 命令名 # 搜索命令所在路径及别名

PATH环境变量

定义的是系统搜索命令的路径
echo $PATH # 显示路径

find命令

find [搜索范围] [搜索条件]
搜索条件

-name：根据名称搜索
-iname：不区分大小写

例如
find / -name install.log
注意

  1. 避免大范围搜索，会非常耗费系统资源
  2. find是在系统当中搜索符合条件的文件名。如果需要匹配，使用通配符匹配，通配符是完全匹配

Linux中的通配符

*    匹配任意字符
？    匹配任意一个字符
[]    匹配任意一个中括号内的字符

find /root -user root # 在/root下搜索root用户的文件
find /root -nouser # 在/root下搜索没有所有者的文件
find /var/log/ -mtime +10 # 查找10天前改的文件

atime  文件访问时间 access time
ctime  改变文件属性时间 change time
mtime  修改文件内容时间 modify time

-10  10天内修改的文件
10  10天当天修改的文件
+10  10天前修改的文件

find . -size 25k # 在当前目录下查找文件大小是25k的文件
find . -size 25M # 在当前目录下查找文件大小是25M的文件
find . -size 25 # 在当前目录下查找文件大小是25个扇区的文件

-25k  小于25KB的文件
25k  等于25kKB的文件
+25k  大于25KB的文件

find . -inum 262422 # 在当前目录下查找节点是262422的文件
find /etc -size +20k -a -size -50k # 查找/etc目录下，大于20k并且小于50k的文件

-a：and 逻辑与
-r：or 逻辑或

find /etc -size +20k -a -size -50k -exec ls -lh {} ;

# 查找/etc/目录下，大于20k并且小于50k的文件，并显示详细信息
# -exec  第二命令  {} \;  对搜索结果执行第二命令,这里是执行ls命令
搜索字符串命令grep

grep [选项] 字符串 文件名 # 在文件当中匹配符合条件的字符串

-i：忽略大小写
-v：排除指定字符串
find命令与grep命令的区别

find：在系统当中搜索符合条件的文件名，如果需要匹配，使用通配符匹配，通配符是完全匹配。
grep：在文件当中搜索符合条件的字符串，如果需要匹配，使用正则表达式进行匹配，正则表达式是包含匹配。

帮助命令

帮助命令man

man 命令 # 获取指定命令的帮助
man -f 命令 # 查看命令拥有哪个级别的帮助，相当于whatis 命令
man -k 命令 # 查看和命令相关的所有帮助，相当于apropos 命令

选项帮助

命令 –help # 获取命令选项的帮助

shell内部命令帮助

help shell内部命令 # 获取shell内部命令的帮助
例如
whereis cd # 确定cd是否是shell内部命令（无命令执行文件）
help cd # 获取cd命令帮助

详细命令帮助info

info 命令

- 回车：进入子帮助页面（带有*号标记）
- u：进入上层页面
- n：进入下一个帮助小节
- p：进入上一个帮助小节
- q：退出

压缩与解压缩命令

常用压缩格式： .zip .gz .bz2 .tar.gz .tar.bz2

.zip格式压缩

zip 压缩文件名 源文件 # 压缩文件
zip -r 压缩文件名 源文件 # 压缩目录

.zip格式解压缩

unzip 压缩文件 # 解压缩.zip文件

.gz格式压缩

gzip 源文件 # 压缩为.gz格式的压缩文件，源文件会消失
gzip -c 源文件 > 压缩文件 # 压缩为.gz格式，源文件保留
例如
gzip -c cangls > cangls.gz
gzip -r 目录 # 压缩目录下所有的子文件，但是不能压缩目录

.gz格式解压缩

gzip -d 压缩文件 # 解压缩文件
gunzip 压缩文件 # 解压缩文件

.bz2格式压缩

bzip2 源文件 # 压缩为.bz2格式，不保留原文件
bzip2 -k 源文件 # 压缩之后保留源文件
注意:bzip2命令不能压缩目录

.bz2格式解压缩

bzip2 -d 压缩文件 # 解压缩，-k保留压缩文件
bunzip2 压缩文件 # 解压缩，-k保留压缩文件

打包命令tar

先用tar对文件打包，再将打包文件压缩成gz或bz2格式
tar -cvf 打包文件名 源文件

-c：打包
-v：显示过程
-f：指定打包后的文件名

例如
tar -cvf longzls.tar longzls

解打包命令

tar -xvf 打包文件名

-x：解打包

例如
tar -xvf longzls.tar

.tar.gz压缩格式 # 先打包为.tar格式，再压缩为.gz格式

tar -zcvf 压缩包名.tar.gz 源文件 # 压缩打包

-z：压缩为.tar.gz格式

tar -zxvf 压缩包名.tar.gz # 解压缩

-x：解压缩.tar.gz格式
.tar.bz2压缩格式

tar -jcvf 压缩包名.tar.bz2 源文件 # 压缩打包

-z：压缩为.tar.bz2格式

tar -jxvf 压缩包名.tar.bz2 # 解压缩

-x：解压缩.tar.bz2格式

tar -jxvf 压缩包名.tar.bz2 -C 指定目录 # 解压到指定位置

关机与重启命令

shutdown命令

shutdown [选项] 时间
选项：

-c：取消前一个关机命令
-h：关机
-r：重启
其他关机命令

halt
poweroff
init 0

其他重启命令

reboot
init 6

系统运行级别
  • 0 关机
  • 1 单用户
  • 2 不完全多用户，不含NFS服务
  • 3 完全多用户
  • 4 未分配
  • 5 图形界面
  • 6 重启

runlevel # 查询系统运行级别

退出登录命令

logout

挂载命令

查询与自动挂载

mount # 查询系统中已经挂载的设备
依据配置文件/etc/fstab的内容，自动挂载

挂载命令格式

mount [-t 文件系统] [-o 特殊选项] 设备文件名 挂载点

-t 文件系统：加入文件系统类型来制定挂载的类型，可以ext3、ext4、iso9660（光盘默认）等文件系统
-o 特殊选项：可以指定挂载的额外选项
挂载光盘

mkdir /mnt/cdrom/ # 建立挂载点
mount -t iso9660 /dev/cdrom /mnt/cdrom/ # 挂载光盘
mount /dev/sr0 /mnt/cdrom/

光盘卸载命令

umount 设备文件名或挂载点
umount /mnt/cdrom or umount /dev/sr0

挂载U盘

fdisk -l # 查看U盘设备文件名（/dev/sdb1）
mount -t vfat /dev/sdb1 /mnt/usb/
注意: Linux默认是不支持NTFS文件系统的

用户登录查看命令

查看用户登录信息

w [用户名]
who [用户名]

查询当前登录和过去登录的用户回信息

last
last命令默认是读取/var/log/wtmp文件数据

查看所有用户的最后一次登陆时间

lastlog
last命令默认是读取/var/log/lastlog文件数据

shell基础

echo输出命令
echo [选项] [输出内容]

Bash（shell的一种）的基本功能

命令别名与快捷键

查看与设定别名
alias # 查看系统中所有的命令别名
alias 别名=’原命令’ # 设定命令别名（重启失效）
别名永久生效
写入环境变量配置文件，例如~/.bashrc
删除别名
unalias 别名 # 删除别名
命令生效顺序

  1. 第一顺位执行用绝对路径或相对路径执行的命令
  2. 第二顺位执行别名
  3. 第三顺为执行Bash的内部命令
  4. 第四顺位执行按照$PATH环境变量定义的目录查找顺序找到的第一个命令
常用快捷键

ctrl+c 强制终止当前命令
ctrl+l 清屏
ctrl+a 光标移动到命令行首
ctrl+e 光标移动到命令行尾
ctrl+u 从光标所在位置删除到行首
ctrl+z 把命令放入后台
ctrl+r 在历史命令中搜索

历史命令

history [选项] [历史命令保存文件]
选项

-c：清空历史命令
-w：把缓存中的历史命令写入历史命令保存文件~/.bash_history

历史命令的调用

  • 上、下箭头调用以前的历史命令
  • 使用”!n”重复执行第n条历史命令
  • 使用”!!”重复执行上一条命令
  • 使用”!字串”重复执行最后一条以该字串开头的命令
输出重定向

标准输出重定向
命令 > 文件 # 覆盖
命令 >> 文件 # 追加
标准错误输出重定向(左右无空格)
错误命令 2>文件 # 覆盖
错误命令 2>>文件 # 追加
正确输出和错误输出同时保存
命令 > 文件 2>&1 # 覆盖
命令 >> 文件 2>&1 # 追加
命令 &> 文件 # 覆盖
命令 &>> 文件 # 追加
命令>>文件1 2>>文件2 # 追加至不同文件（正确1 错误2）

多命令顺序执行

; 命令1;命令2 顺序执行 # 半角分号
&& 命令1&&命令2 逻辑与
|| 命令1||命令2 逻辑或
命令 && echo yes || echo no # 判断命令是否正确执行

管道符

命令1 | 命令2 # 命令1的正确输出作为命令2的操作对象，可嵌套运用
e.g.
ll -a /etc/ | more
netstat -an | grep “EATABLISHED”

软件安装

yum命令

常用yum命令

查询
yum list # 查询所有可用软件包列表
yum search 关键字 # 搜索服务器上所有和关键字相关的包
安装
yum -y install 包名

- y 自动回答yes

升级
yum -y update 包名
卸载
yum -y remove 包名
服务器使用最小化安装，用啥装啥，尽量不卸载

RPM安装

安装
rpm -ivh 包全名

-i （install） 安装
-v （verbose） 显示详细信息
-h （hash） 显示进度
--nodeps不检测依懒性

RPM包升级
rpm -Uvh 包全名（新版本的rpm包）

-U（upgrade） 升级

卸载
rpm -e 包名

-e（erase）卸载
--nodeps不检查依赖性

RPM包查询

查询是否安装

rpm -q 包名

-q （query） 查询

rpm -qa

-a （all） 所有
查询软件包详细信息

rpm -qi 包名

-i （informatinn） 查询软件信息
-p （package） 查询未安装包信息
查询包中文件安装位置

rpm -ql 包名

-l （list）列表
-p （package） 查询未安装包信息
查询系统文件属于哪个RPM包

rpm -qf 系统文件名

-f （file）查询系统文件属于哪个软件包
查询软件包的依赖性

rpm -qR 包名

-R （requires） 查询软件包的依赖性
-p （package） 查询未安装的包信息

RPM包校验

RPM包校验
rpm -V 已安装的包名

-V （verify） 校验指定RPM包中的文件
